# DAWcicle — Tienda de Bicicletas
Sistema de gestión de una tienda de bicicletas: inserción de productos (bicicletas y artículos), compra con sistema de descuentos acumulativos por categoría, y control de stock. Usa clases abstractas y excepciones personalizadas.
**Conceptos:** `abstract`, `ArrayList`, excepciones custom, herencia, menú interactivo, descuentos.
```bash
cd src && javac *.java && java DAWcicle
```
