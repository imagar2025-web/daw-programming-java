# Destruir la Flota (Hundir la Flota)
Juego completo de Hundir la Flota con IA para colocar y disparar barcos. Incluye mapa 10x10, sistema de coordenadas, tipos de barco (velero, fragata, buque, portaviones), excepciones personalizadas y turnos jugador vs máquina.
**Conceptos:** POO completa (herencia con Mapa abstracto), `ArrayList`, excepciones custom, IA básica, `Random`.
```bash
cd src && javac *.java && java App
```
