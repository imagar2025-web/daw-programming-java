# MasterMind
Juego de MasterMind por consola con selección de dificultad (fácil/media/difícil), generación aleatoria de combinaciones de colores y sistema de vidas.
**Conceptos:** `Random`, bucles de juego, validación de entrada, lógica condicional compleja.
```bash
javac src/MasterMindDAW1.java && java -cp src MasterMindDAW1
```
