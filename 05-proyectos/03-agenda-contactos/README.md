# Agenda de Contactos
Aplicación de gestión de contactos con sistema de usuarios (login/registro), persistencia con serialización, validación de email y teléfono, y menú interactivo CRUD.
**Conceptos:** `Serializable`, `ArrayList`, `FileInputStream/OutputStream`, validación, sistema multiusuario.
```bash
cd src && javac *.java && java Main
```
