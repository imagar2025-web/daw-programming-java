# Sistema de Gestión de Trenes
Aplicación de gestión de destinos de trenes con registro de usuarios, venta de billetes, destinos de cercanías y largas distancias. Implementa clases abstractas y herencia.
**Conceptos:** `abstract`, herencia, `ArrayList`, sistema de usuarios, `toString` sobreescrito.
```bash
cd src && javac *.java && java App
```
