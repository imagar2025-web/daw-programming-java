# Bucles
Ejercicios con `for`, `while` y `do-while`. Incluye generación de figuras geométricas con bucles anidados y funciones auxiliares con arrays.
**Conceptos:** `for`, `while`, `do-while`, bucles anidados, métodos estáticos.
```bash
javac src/DibujosConBucles.java && java -cp src DibujosConBucles
```
