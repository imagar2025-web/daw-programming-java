# 01 — Fundamentos de Java
Primeros pasos con Java: desde variables y tipos hasta funciones, pasando por condicionales, bucles, arrays y strings. Cada carpeta aborda un concepto fundamental.

| Carpeta | Tema |
|---------|------|
| `01-variables-y-tipos` | Tipos primitivos, Scanner, constantes |
| `02-condicionales` | if-else, switch, validación |
| `03-bucles` | for, while, do-while, bucles anidados |
| `04-arrays` | Arrays, búsqueda, rotación |
| `05-strings` | Manipulación de cadenas |
| `06-funciones` | Métodos estáticos, modularización |
