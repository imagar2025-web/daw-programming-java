# Arrays
Operaciones sobre arrays: suma, media, clasificación par/impar, búsqueda de máximo/mínimo con posiciones, rotación de elementos y conteo de caracteres.
**Conceptos:** arrays unidimensionales, recorrido, búsqueda, manipulación de posiciones.
```bash
javac src/RotacionArray.java && java -cp src RotacionArray
```
