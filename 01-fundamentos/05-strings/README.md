# Strings
Manipulación de cadenas: conversión a ASCII con `StringBuilder`, comprobación de subcadenas con `startsWith`, métodos `replace`, `substring`, `toUpperCase`.
**Conceptos:** `String`, `StringBuilder`, métodos de cadena, `charAt`, recorrido de strings.
```bash
javac src/StringToAscii.java && java -cp src StringToAscii
```
