# Condicionales
Ejercicios con `if-else`, `switch` y condicionales anidados aplicados a problemas reales: validación de fechas, clasificación de triángulos, conversión de monedas y cálculo de cuotas con descuentos.
**Conceptos:** `if-else`, `switch-case`, operadores lógicos, validación de entrada.
```bash
javac src/ConversorMonedas.java && java -cp src ConversorMonedas
```
