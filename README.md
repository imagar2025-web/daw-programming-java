## 📈 Progresión

El contenido sigue el orden cronológico del módulo:

**Bloque 1** → Fundamentos: tipos, condicionales, bucles, arrays, strings y funciones.

**Bloque 2** → POO: clases, composición, herencia, polimorfismo, abstract e interfaces.

**Bloque 3** → Excepciones y estructuras de datos avanzadas.

**Bloque 4** → Acceso a ficheros y serialización.

**Bloque 5** → Proyectos integradores que combinan todo lo anterior.

## 🚀 Cómo ejecutar

```bash
# Clonar el repositorio
git clone https://github.com/imagar2025-web/daw-programming-java.git

# Navegar a cualquier ejercicio
cd daw-programming-java/05-proyectos/02-hundir-la-flota/src

# Compilar y ejecutar
javac *.java
java App
```
