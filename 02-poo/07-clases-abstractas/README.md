# Clases Abstractas — Productos
Clase abstracta `Producto` con implementaciones concretas: `ProductoElectronico` (TV, MP3) y `Libro`. Demuestra que no se puede instanciar una clase abstracta y cómo se fuerza la implementación de métodos.
**Conceptos:** `abstract`, métodos abstractos, jerarquía de clases, contratos de implementación.
